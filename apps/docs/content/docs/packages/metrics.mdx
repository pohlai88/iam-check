## Package README

**What it is** тАФ Rank-1 Platform Prometheus metrics leaf for Afenda-Lite: dedicated registry, predeclared HTTP/DB/cache instruments, and scrape text rendering via `prom-client`.

**What it does** тАФ Builds an isolated (or process-default) registry, optionally collects Node default metrics, records low-cardinality counters/histograms, and renders Prometheus exposition text for a scrape endpoint.

**When you need it** тАФ Node Route Handlers or platform adapters that must emit aggregate RED-style metrics without inventing a vendor APM / OpenTelemetry stack.

**Who it's for** тАФ `apps/web` (scrape RH + route-pipeline recording). Next-free leaf: no `@afenda/*` runtime deps, no Pages/Prisma wrappers, no org-id labels.
